# Dataset Card

## 数据来源

数据来自个人 HKSYU 邮箱的私有归档，经筛选、匿名化、风险审计和知识化后发布。本仓库只包含派生知识，不包含原始邮件。

## 处理流程

1. 从私有 Outlook/PST 导出中筛选 HKSYU 相关内容。
2. 将重复邮件和相近主题归并为事件卡。
3. 移除个人身份、私有链接、本机路径和高风险上下文。
4. 只保留低风险事件卡、主题指南、年月时间线、FAQ 和统计数据。

## 字段说明

`data/events.public.jsonl` 每行是一个事件卡，字段包括事件 ID、标题、年份范围、月份、类别、部门、行动类型、人类摘要、AI 摘要、常见时间点、公开链接、来源数量、置信度、风险等级和对应 Markdown 路径。

## 已知限制

- 这不是 HKSYU 官方校历或官方政策文件。
- 邮件样本来自个人邮箱，存在个人经历偏差。
- 为保护隐私，部分细节被概括或删除。
- 学校政策会变化，读者应以 HKSYU 官方网页和当年通知为准。
