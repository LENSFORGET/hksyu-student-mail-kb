# HKSYU Student Mail Knowledge Base

这是一个由个人大学邮箱派生出的 HKSYU 学习生活知识库。它面向后来者阅读、校正和检索，也为 AI/RAG 系统提供低风险结构化语料。

- 资料总体时间段：2022年08月01日 至 2026年05月08日
- 覆盖年份：2022-2026
- 私有源记录规模：2676 条学校相关邮件/事件线索
- 当前公开事件卡：893
- 公开形式：主题指南、年月时间线、FAQ、事件卡和统计表
- 浏览入口：GitHub Markdown 与 GitHub Pages 三栏浏览器
- 项目状态：非官方、社区维护、由邮件资料匿名化和知识化后生成

> 说明：当前公开仓接入的匿名化中间层覆盖到 2026年05月08日。原始主 PST 已知仍有 2026年06月邮件，待重新导出、匿名化和验证后再补入公开仓。

## 项目说明

本项目不是 HKSYU 官方资料库，也不是个人邮箱镜像。它把大学邮箱中与 HKSYU 学习生活有关的通知、系统提醒、课程信息、学生事务和职业发展线索整理成可阅读的知识库。公开版本只保留低风险的派生摘要和事件卡，便于后来者按时间、主题、部门和关键词查阅。

数据源经过筛选和匿名化处理：原始邮件、邮件头、联系人、附件、日历、任务和个人身份信息均不进入本仓库。公开内容适合用于经验查阅、人工校正和 AI/RAG 检索，但不应用作官方政策依据。

## 如何阅读

1. 想按学期或月份查找事项，请从 [年度时间线](knowledge/timeline/) 开始。
2. 想按部门或系统查找事项，请进入 [主题指南](knowledge/topics/)。
3. 想快速理解常见问题，请看 [常见问题](knowledge/faq/)。
4. 想用类似 Outlook 的方式浏览，请打开 [GitHub Pages 浏览器](https://lensforget.github.io/hksyu-student-mail-kb/)。
5. 想给 AI 或脚本使用，请读取 [事件卡数据](data/events.public.jsonl)。

## 不包含什么

本仓库不公开 PST、MSG、邮件全文、联系人、日历、任务、附件原件、个人邮箱、学生证号、付款编号、申请编号、成绩、推荐信、投诉、申诉、医疗或辅导相关内容。
