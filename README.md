# HKSYU Student Mail Knowledge Base

这是一个由个人大学邮箱派生出的 HKSYU 学习生活知识库。它面向后来者阅读、校正和检索，也为 AI/RAG 系统提供低风险结构化语料。

- 覆盖范围：约 2022-2025
- 当前公开事件卡：319
- 公开形式：主题指南、年月时间线、FAQ、事件卡和统计表
- 浏览入口：GitHub Markdown 与 GitHub Pages 三栏浏览器
- 项目状态：非官方、社区维护、由邮件资料匿名化和知识化后生成

## 快速入口

- [年度时间线](knowledge/timeline/)
- [主题指南](knowledge/topics/)
- [常见问题](knowledge/faq/)
- [事件卡数据](data/events.public.jsonl)
- [GitHub Pages 浏览器](docs/)

## 不包含什么

本仓库不公开 PST、MSG、邮件全文、联系人、日历、任务、附件原件、个人邮箱、学生证号、付款编号、申请编号、成绩、推荐信、投诉、申诉、医疗或辅导相关内容。

如发现隐私问题，请优先按 [PRIVACY.md](PRIVACY.md) 提交移除请求。
